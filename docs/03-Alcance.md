# Alcance

## Incluye

- API Experience.
- API Support.
- Autenticacion JWT en API Experience.
- Validacion de usuario mediante GoREST.
- Registro de la informacion solicitada para cada operacion.
- Registro, actualizacion y busqueda en API Support.
- Persistencia reactiva con R2DBC.
- MySQL como configuracion de ejecucion.
- H2 mediante el perfil `h2` para pruebas.
- Manejo de excepciones.
- Programacion reactiva.
- Swagger.

## Dependencias externas

- Servicio GoREST.
- Cliente HTTP como Postman, SoapUI o frontend.
- Motor MySQL para ejecucion.
- Motor H2 para pruebas.
